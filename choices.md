# 개인정보 선택 사항 (Privacy Choices)

흘러(Flow) 이용자분께서는 본인의 개인정보에 대해 다음과 같은 권리를 행사하실 수 있습니다.

## 데이터 열람 및 관리

본 앱은 별도의 사용자 계정이나 서버 데이터베이스를 운영하지 않습니다. 모든 데이터(위치 캐시, 즐겨찾기, 언어·알림 설정)는 이용자의 기기 내에만 저장됩니다.

## 데이터 삭제 방법

원하시는 데이터를 즉시 삭제하실 수 있습니다.

### 1. 위치 정보 캐시 삭제
- iOS 설정 → 개인정보 보호 및 보안 → 위치 서비스 → 흘러 → "허용 안 함" 선택
- 마지막 매칭 결과가 즉시 삭제되며, 재실행 시 다시 캐시되지 않습니다.

### 2. 광고 식별자 사용 중지
- iOS 설정 → 개인정보 보호 및 보안 → 추적 → 흘러 → 비활성화
- 또는 iOS 설정 → 개인정보 보호 및 보안 → 추적 → "앱이 추적 요청 허용" 전체 비활성화
- 비활성화 즉시 IDFA가 0으로 마스킹되며, 비맞춤형 광고만 표시됩니다.

### 3. 즐겨찾기 / 설정 데이터 삭제
- 앱 내 즐겨찾기 항목을 직접 해제하거나
- 앱을 삭제하시면 모든 로컬 데이터가 즉시 삭제됩니다.

### 4. 모든 데이터 일괄 삭제
- 홈 화면에서 흘러 앱 길게 누르기 → "앱 제거" → "앱 삭제"
- 기기 외부에 저장된 데이터가 없으므로 앱 삭제만으로 모든 정보가 완전히 삭제됩니다.
