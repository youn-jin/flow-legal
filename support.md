# 흘러(Flow) 고객지원

흘러를 이용해주셔서 감사합니다.

## 자주 묻는 질문

### Q. 우리 동네 수돗물 카드가 안 떠요.
위치 권한이 켜져 있는지 확인해주세요.

iOS 설정 → 개인정보 보호 및 보안 → 위치 서비스 → 흘러 → "앱을 사용하는 동안" 선택해주시면 자동으로 매칭됩니다.

### Q. 부산·광주·대구에서 측정소 정보가 안 나와요.
측정소 단위 시간별 수질 데이터는 현재 서울특별시만 공개하고 있어, 그 외 지역에서는 정수장 단위 데이터로 안내드립니다. 지도 탭에서 가까운 K-water 정수장의 실시간 수질을 확인하실 수 있습니다.

### Q. 광고를 없애고 싶어요.
설정 탭 → "광고 제거" 인앱 구매로 영구 제거 가능합니다. 기기를 변경하셨거나 앱을 재설치하신 경우 "구매 복원"을 누르시면 됩니다.

### Q. 데이터가 너무 오래된 것 같아요.
본 앱은 공공기관이 발표한 시점의 데이터를 가공 없이 그대로 보여드립니다. 일부 정수장은 발표 주기가 길거나, 공공 API 응답이 지연되는 경우가 있습니다.

### Q. 측정값을 의료적 판단의 근거로 사용해도 되나요?
공공기관이 공개한 모니터링 데이터를 안내해드리는 정보 제공 앱입니다. 의료·과학적 진단의 근거로는 사용하지 마시고, 식수 안전과 관련해 의문이 있으시면 거주 지역 상수도사업본부에 직접 문의해주세요.

### Q. 즐겨찾기가 사라졌어요.
즐겨찾기는 기기 로컬에만 저장되며 클라우드 동기화는 하지 않습니다. 앱을 삭제하시거나 기기를 변경하시면 함께 삭제됩니다.
